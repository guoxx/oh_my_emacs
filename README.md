oh_my_emacs
===========

## Requirements

### modes
* nginx-mode
* protobuf-mode
* cmake-mode
* markdown-mode

### utils
* browse-kill-ring
* undo-tree
* starter-kit
* helm
* helm-gtags
* frame-cmds
* frame-fns
* direx
* exec-path-from-shell
* find-file-in-project
* multiple-cursors

## Installation
create directory userdata in root emacs config directory(for example .emacs.d/userdata), variable `userdata-dir` was defined in dot-emacs.
