oh_my_emacs
===========

## Requirements

### modes
* nginx-mode
* protobuf-mode
* cmake-mode
* markdown-mode

### utils
* undo-tree
* starter-kit
* helm
* helm-gtags
* frame-cmds
* frame-fns
* direx
* exec-path-from-shell
* multiple-cursors
* yasnippet
* yasnippet-bundle
* window-jump
* windresize

## Installation
create directory userdata in root emacs config directory(for example .emacs.d/userdata), variable `userdata-dir` was defined in dot-emacs.
